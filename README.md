# GhostBuster
GhostBusters adalah game platformer 2D yang dibuat menggunakan Python dengan library Tkinter. Pada game ini pemain mengendalikan karakter hantu yang harus melewati berbagai platform, mengumpulkan koin, dan mencapai tujuan akhir pada setiap stage. Tampilan game dibuat dengan tema malam hari yang menampilkan bulan besar, langit malam dan platform.
