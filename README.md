# GhostBuster
GhostBusters adalah game platformer 2D yang dibuat menggunakan Python dengan library Tkinter. Pada game ini pemain mengendalikan karakter hantu yang harus melewati berbagai platform, mengumpulkan koin, dan mencapai tujuan akhir pada setiap stage. Tampilan game dibuat dengan tema malam hari yang menampilkan bulan besar, langit malam dan platform.

Langkah Pengoperasian
1. Register, buat akun baru dengan username dan password (minimal 4 karakter), atau Login jika sudah punya akun.
2. Setelah masuk, akan berada di Menu Utama: Mulai Game, Leaderboard, Pengaturan, Keluar.
3. Pilih Mulai Game untuk memulai dari Level 1. Jika sebelumnya pernah keluar di tengah permainan, akan muncul pilihan Lanjut dari level terakhir atau Ulang dari Level 1.
4. Capai garis finish sebelum waktu habis untuk lanjut ke level berikutnya. Kalah jika nyawa habis (3 nyawa), waktu habis sebelum finish.
5. Setelah menyelesaikan level terakhir (menang) atau kalah, muncul layar Game Over/Menang dengan skor akhir. Skor otomatis tersimpan ke database jika menang.
6. Cek Leaderboard untuk melihat peringkat skor tertinggi, atau Pengaturan untuk melihat skor terbaik dan level tertinggi.

Fitur
1. 3 level dengan tingkat kesulitan berbeda
2. Physics platformer, tembak-menembak, koin, dan efek partikel
3. Skor, bonus waktu, dan sistem nyawa
4. Pause/resume dan simpan progres level
5. Login/register, leaderboard, dan skor terbaik (MySQL)
